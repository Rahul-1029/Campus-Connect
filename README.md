Campus Connect is a localized marketplace designed specifically for the MVGR student community. It facilitates the buying, selling, and gifting of academic essentials like textbooks, engineering tools (drafters, lab coats), and electronics.

Instead of letting expensive resources sit idle after a semester, students can list them for juniors to find easily, promoting sustainability and affordability on campus.

Real-time Marketplace: Browse live listings from fellow students powered by Google Firebase.

Secure Management: Sellers use a unique 4-digit PIN to manage and delete their listings, preventing unauthorized changes.

One-Click Communication: Direct WhatsApp integration allows buyers to message sellers instantly with pre-filled item details.

Optimized Web Experience: Custom-built Flutter web interface optimized for mobile and desktop browsers with fast-loading HTML rendering.

Branded Experience: Designed with a clean Material 3 UI utilizing the official GDG color palette.
